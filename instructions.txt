```

```

---

# ⚙️ 3. Backend Setup (Node.js + Express)

## 📌 Step 1: Navigate to backend

```
cd backend
```

---

## 📌 Step 2: Install dependencies

```
npm install
```

---

## 📌 Step 3: Create `.env` file

Create a file named `.env` in backend root:

```
DB_HOST=your_host
DB_USER=your_username
PASSWORD=your_password
DB_NAME=your_database
JWT_SECRET=secretkey
```

---

## 📌 Step 4: Setup Database

Run this SQL in MySQL:

```
CREATE TABLE auth_user (
    id INT AUTO_INCREMENT PRIMARY KEY,
    email VARCHAR(255) NOT NULL UNIQUE,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    password VARCHAR(255) NOT NULL
);

CREATE TABLE teachers (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL UNIQUE,
    university_name VARCHAR(150) NOT NULL,
    gender ENUM('male','female','other') NOT NULL,
    year_joined YEAR NOT NULL,
    FOREIGN KEY (user_id) REFERENCES auth_user(id)
        ON DELETE CASCADE
);
```

# 📡 API Summary

| Method | Endpoint             | Description        |
| ------ | -------------------- | ------------------ |
| POST   | /api/auth/login      | Login              |
| POST   | /api/teachers/create | Create teacher     |
| GET    | /api/teachers        | Get all teachers   |
| GET    | /api/teachers/:id    | Get single teacher |

---

# ⚠️ Important Notes

* Backend must run before frontend
* JWT token is required for protected APIs
* CORS must be enabled in backend
* `.env` file should not be shared publicly

---

# 🧠 Troubleshooting

### ❌ "No token"

→ Login first

### ❌ "ECONNREFUSED"

→ Backend not running

### ❌ "CORS error"

→ Enable cors middleware

### ❌ Empty data

→ No records in database
